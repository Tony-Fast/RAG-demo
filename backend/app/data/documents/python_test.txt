Python 面试常见问题汇总

1. Python 是什么语言？
Python 是一种解释型、面向对象、动态数据类型的高级程序设计语言。

2. Python 的特点有哪些？
- 简单易学
- 语法简洁
- 开源免费
- 跨平台
- 丰富的库支持

3. 什么是列表推导式？
列表推导式是 Python 的一种简洁创建列表的方式。

4. Python 中的字典是什么？
字典是一种键值对的数据结构，用于存储映射关系。

5. 什么是装饰器？
装饰器是 Python 中用于修改函数行为的特殊函数。

6. Python 有哪些数据类型？
- 整型 (int)
- 浮点型 (float)
- 字符串 (str)
- 列表 (list)
- 字典 (dict)
- 布尔型 (bool)

7. 什么是虚拟环境？
虚拟环境是 Python 中用于隔离项目依赖的工具。

8. Python 2 和 Python 3 的区别？
Python 3 是 Python 2 的升级版本，语法更规范，不再兼容 Python 2。
